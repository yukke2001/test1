# 🎯 Phase 3: ゲーム機能統合 実装計画

## Phase 3 の目標
認証システム（Phase 2）と既存のゲーム機能を統合し、ユーザー別のゲーム記録管理システムを構築する。

## 実装ステップ

### Step 1: ゲーム認証統合 (30分)
- [ ] game.jsp に認証状態チェック機能を追加
- [ ] ログイン済みユーザー情報をゲーム画面に表示
- [ ] ゲスト/認証ユーザーの分岐処理実装

### Step 2: ゲーム記録保存機能 (45分)
- [ ] GameRecordDAO の拡張・修正
- [ ] ゲーム完了時のスコア保存機能
- [ ] ユーザー別統計情報の蓄積

### Step 3: ランキング・統計表示 (30分)
- [ ] home.jsp のランキング機能実装
- [ ] 個人統計情報の表示
- [ ] 全体ランキングの表示

### Step 4: 統合テスト・最適化 (15分)
- [ ] 認証→ゲーム→記録保存の完全フロー確認
- [ ] UI/UX の調整
- [ ] エラーハンドリングの強化

## 期待される成果
- 完全に統合された認証付きメモリーカードゲーム
- ユーザー別の進捗追跡
- 競争要素のあるランキングシステム
- データベース駆動型のユーザーエクスペリエンス

## Phase 3 完了条件
1. ログインユーザーがゲームを完了し、記録が保存される
2. ランキング機能が正常に動作する
3. 認証なしでもゲストプレイが可能
4. データベースにゲーム記録が正しく蓄積される

## 推定所要時間: 約2時間
